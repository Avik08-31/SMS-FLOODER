## Usage

###### Update the packages

* Termux

pkg up -y


* Kali Linux/Parrot OS

apt update -y && apt upgrade -y


##### Clone the repository


git clone https://github.com/Avik080/SMS-FLOODER


    
###### Go to the Xlr8 directory

>> cd SMS-FLOODER

###### Give permission

>> chmod +x SMS FLOODER

###### Run the script

>> bash xlr8.sh
